# aos-central-playground

Sandbox to prototype AOS CENTRAL SURFACE with GitHub (and later Linear).

## Purpose

- Capture the core concepts of AOS CENTRAL SURFACE (lanes, briefs, consoles, grants, sims).
- Define how work moves from AOS → issues → branches → pull requests.
- Serve as a safe place to practice GitHub basics (commits, branches, PRs).

## Repo structure (v0)

- /docs – AOS concepts and flows
- /protocols – Naming rules, branch/PR conventions
- /notes – Scratchpad for ideas and experiments
